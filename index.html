Dynamic Traffic Lights

Runner up in Manipal Hackathon 2019

Won a runner up spot in Manipal Hackathon 2019. The hackathon was about social innovation in which I developed a Smart Dynamic Traffic Light System which uses Deep Learning to assess traffic jams at junction in real time and control the flow and timer of the signal accordingly.

  • What is it for?
    • Our solution is for addressing traffic problems in the country, the goal is to make traffic management efficient by reducing long waiting times and wastage of green signal time.
  • What will it do?
    • Our end solution once implemented it will make an ordinary traffic light system into a smart efficient traffic management system at traffic signals which can guide and control traffic not by fixed timers but dynamically acting by analyzing the situations on every lane.
  • Which are the potential markets?
    • Global Traffic Management market
  • Unique features?

    • Dynamic signal is a signal that will assess the conditions and will accordingly set a time for that particular lane.Thus dynamically changing waiting time which will surely positively impact on traffic conditions.

    Timers of the traffic signal will be set not only on the factor of count but also various factors such as vehicles type, size, acceleration, head-space, width of that road etc.

    Along with the timer we will also control the flow of the traffic efficiently by using our own SLRL algorithm implemented into our system.

Warning System for Driver

Shortlisted in SIH 2020 finale

Day by day vehicles are increasing on the road. So as the safety concerns should increase. Risk of accidents due to over speeding vehicles in low speed regions is also increasing exponentially. Often the safety of the driver as well as passengers is compromised. Even the safety of school areas as well as hospital areas is compromised to a certain extent. To overcome this problem a system is to be developed know as Warning System for Driver. With the help of this system we will warn the driver about exceeding the speed limit.

Also, we will give an alert about the same to the driver. Alert can be in any form -sound, voice etc. This has to be done in a way that it does not create privacy issue for the customer. Even the data collected from this system can be used as a crowd source data for better implementation. And for increasing the accuracy of the system maps API is also integrated.
